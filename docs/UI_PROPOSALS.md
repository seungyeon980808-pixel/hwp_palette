# UI 개선 제안 20선 (중요도 순)

2026-07-19, 전체 UI 를 훑고 뽑은 목록. **중요도 = "사용자가 실제로 겪을 손해·불편의
크기 ÷ 구현 비용"** 기준. 위에서부터 하는 것을 권하지만 순서는 조정 가능.

> 표기: 🔴 손실 방지 · 🟠 눈에 보이는 불편 · 🟡 효율 · 🔵 완성도

---

### 1. 🔴 팔레트 편집 실행 취소 (Ctrl+Z)
환경설정에서 블럭을 잘못 삭제하면 **복구할 방법이 없다**. 서식 조합은 다시 만들려면
값들을 기억해야 한다. 편집 직전의 blocks 목록을 몇 단계 저장해 두면 끝 — 저장소가
JSON 이라 스냅샷 비용이 거의 0이다.

### 2. 🔴 config.json / library.json 자동 백업
파일이 깨지면(디스크 오류, 강제 종료 중 저장) 팔레트·라이브러리가 통째로 사라진다.
저장할 때마다 `.bak1~3` 롤링 백업 — 코드 몇 줄로 최악의 사고를 막는다.

### 3. 🟠 상태 표시줄 개선 — 색 구분 + 최근 알림 열람
성공(✅)·경고(⚠)가 같은 회색 한 줄이라 놓치기 쉽고, 다음 메시지가 오면 사라진다.
성공=초록/경고=주황/오류=빨강 배경 + 클릭하면 최근 알림 10개 목록. app.log 인프라가
이미 있어 붙이기 쉽다.

### 4. 🟠 한글 연결 상태 표시등
제목줄에 초록(연결됨)/회색(안 됨) 점 하나. 지금은 버튼을 눌러 실패해야 연결 여부를
안다. 주기적 가벼운 확인(_connection_error 재사용)으로 누르기 전에 보이게.

### 5. 🟠 변환 미리보기
Ctrl+T 전에 "템플릿 2개 · 빈칸 5개 채움 · 경고 1건"을 한 줄로 보여주고 진행/취소.
파서가 이미 ops/warns 를 만들므로 표시만 붙이면 된다. 잘못된 변환을 실행 전에 잡는다.

### 6. 🟠 블럭 툴팁에 '내용' 미리보기
지금 툴팁은 이름뿐이다. 서식 조합이면 "굵게 + 크기15 + 자간-5", 템플릿이면 빈칸
수·첫 줄 텍스트까지 보여주면 잘못 누르는 일이 줄어든다.

### 7. 🟡 라이브러리 항목 썸네일
템플릿·양식의 실제 모습을 작은 그림으로. **HWPX 안에 한글이 만든 미리보기
(Preview/PrvImage.png)가 이미 들어 있다** — 변환 한 번이면 공짜로 얻는다.
이름만 보고 고르는 현재 방식의 근본 한계를 푼다.

### 8. 🟡 통합 검색 (Ctrl+K)
블럭·라이브러리·내장 문자를 한 창에서 검색해 Enter 로 즉시 실행. 블럭이 수십 개가
되면 격자에서 눈으로 찾는 것보다 훨씬 빠르다.

### 9. 🟡 블럭에 단축키 할당
자주 쓰는 블럭에 Ctrl+1~9 를 배정. 시험 조판처럼 같은 블럭을 수십 번 누르는
작업에서 마우스 왕복이 사라진다.

### 10. 🟡 드래그로 블럭 크기 조절
타일 모서리를 끌어 span×rows 변경. 우클릭 메뉴의 +1/-1 보다 직관적이고,
이미 좌표 수학(_xy_to_cell)이 있어 구현 절반은 돼 있다.

### 11. 🔵 첫 실행 안내
빈 팔레트로 시작하는 구조라 처음엔 뭘 해야 할지 모른다. 블럭이 0개일 때
"① 한글에서 표 선택 → ② 환경설정에서 빈칸 드래그 → ③ 클릭" 3단계 카드 표시.
동료에게 배포할 때 진입 장벽을 없앤다.

### 12. 🔵 환경설정 격자 좌표 눈금
15×N 격자가 되니 "어디에 놓았는지"를 말로 전달하기 어렵다. 열 머리글(1~15)과
드래그 중 상태줄에 "3×2 @ (2,5)" 표시.

### 13. 🔵 블럭 복제 (우클릭 → 복제)
비슷한 서식 조합을 여러 개 만들 때 처음부터 다시 체크하는 대신 복제 후 수정.

### 14. 🔵 탭 드래그 정렬
탭 순서를 ▲▼ 버튼 대신 목록에서 끌어서. 블럭이 이미 드래그인데 탭만 버튼인 건
관성이 어긋난다.

### 15. 🔵 창 위치 기억
지금은 늘 화면 오른쪽 위에 뜬다. 마지막 위치를 config 에 저장 — 멀티 모니터에서
매번 옮기는 수고 제거.

### 16. 🔵 경고 다시 보기
변환 경고창을 닫으면 내용이 사라진다. 상태줄 "⚠ 경고 3건" 클릭 시 마지막 경고
목록을 다시 보여주기 (3번과 묶어 구현 가능).

### 17. 🔵 다크 모드
색 상수가 파일 상단 한 곳에 모여 있어 팔레트 교체 비용이 낮다. 야간 작업용.

### 18. 🔵 키보드 접근성
Tab 포커스 순서 정리 + Enter 실행 + 포커스 표시. 마우스 없이도 모든 기능 접근.

### 19. 🔵 메인 버튼칸 높이 동기화
'메인' 탭 블럭이 여러 줄이 되면 변환 버튼 높이와 어긋날 수 있다. 줄 수에 맞춰
변환 버튼도 늘어나게.

### 20. 🔵 단일 exe 배포 + 업데이트 확인
pyinstaller 로 exe 하나로 묶으면 동료가 파이썬 설치 없이 쓴다. 시작 시 GitHub
release 태그와 비교해 "새 버전 있음"만 알려주기 (자동 업데이트는 하지 않음).
