# exam_scribe

시험문제 이미지(PNG)를 읽어 한글(HWP) 문서에 자동으로 타이핑해주는 도구.

## 목표
시험문제를 편하게 출제하는 것. [5E(phy_draw)](https://github.com/seungyeon980808-pixel/phy_draw)가 그림을 담당한다면, 이 도구는 텍스트·표·수식을 담당한다.

## 구조 (확정, 2026-07-04)
- 순수 로컬 데스크톱 프로그램. 서버/웹 없음.
- 이미지 → Claude API(비전) → 구조화 마크다운(`발문:/자료:/질문:/보기:/선지:`) → pyhwpx(COM)로 한글에 직접 타이핑
- 비전 모델: `claude-sonnet-5` 기본, 파싱 실패 시 `claude-opus-4-8`로 승격

## 파일 구성
- `parser.py` — 마크다운 텍스트 파싱 (순수 함수, 외부 의존성 없음)
- `hwp_engine.py` — 한컴 자동화(pyhwpx) 엔진. 모든 치수·글꼴·테두리를 활성 프리셋에서 읽음. Tkinter/UI에 의존하지 않음
- `settings.py` — 양식 프리셋 저장소 + 빠른 입력 버튼 저장 (config.json)
- `settings_ui.py` — 양식 설정 창 (Toplevel)
- `quickbuttons_ui.py` — 빠른 입력 버튼 편집 창 (Toplevel)
- `main.py` — Tkinter UI, 실행 진입점 (`python main.py` 또는 `run.bat`)
- `config.json` — 사진 폴더 경로 + 양식 프리셋 등 사용자 설정 (git 추적 제외)

## 양식 프리셋
표·박스·글꼴의 모든 스펙을 코드 수정 없이 **프로그램 안 `⚙ 양식 설정`에서** 바꿀 수 있다.
- 학교/시험지별로 양식을 이름 붙여 저장하고 드롭다운으로 전환 (예: `대왕중 2단`, `1단 모의고사`)
- **단 폭 하나(2단 93.99mm ↔ 1단 155mm)를 바꾸면 나머지 표 폭이 자동으로 따라감**
- 글꼴·글자크기(강제 적용 토글), 각 박스 높이, 보기박스 줄간격·셀 여백, 선지 행 높이, 발문/질문 간격, 테두리 종류(없음/실선/점선/이중선)까지 설정
- 📤 내보내기 / 📥 가져오기(JSON)로 다른 선생님과 양식 공유
- 새 설치 시 `기본 (2단)`, `기본 (1단)` 프리셋이 자동 생성됨

## 실행
```
pip install -r requirements.txt
python main.py
```
한글(HWP) 프로그램을 먼저 실행해둘 필요는 없음 — 버튼을 누르면 자동 연결/실행됨.

## 빠른 입력 버튼
자주 쓰는 유니코드 기호(Ω, →, ℃, ², ₁, α, Δ …)를 버튼으로 만들어 클릭 한 번에 삽입한다.
- `빠른 입력` 옆 `✏ 편집` → 격자에서 원하는 칸에 기호를 넣으면 그 위치에 버튼 생성
- 기호를 직접 붙여넣거나 `U+2126` 형식 코드포인트로도 입력 가능 (빈 칸은 버튼 없음)
- 저장은 전역(config.json) — 양식 프리셋과 무관하게 유지됨
