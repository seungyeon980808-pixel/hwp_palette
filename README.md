# hwp_palette

**한글(HWP) 작업 중 "내가 필요한 것"만 골라 자동화하는 도구.**

이 프로그램은 기능이 미리 정해져 있지 않다. **빈 팔레트로 시작해서, 내가 자주 쓰는
것을 내가 원하는 방식으로 등록해 채워 넣는다.** 남이 만들어 둔 메뉴에 나를 맞추는 게
아니라, 내 손버릇에 맞는 도구를 내가 만드는 쪽에 가깝다.

```
내가 자주 쓰는 문자·문구   →  등록해두고 클릭 한 번
내가 만든 표·양식         →  통째로 저장해두고 클릭 한 번
내가 늘 하는 서식 조합     →  하나로 묶어 클릭 한 번
내가 정한 이름으로 호출     →  마크다운으로 쭉 쓰고 한 번에 변환
```

**무엇을 등록할지, 어떤 이름으로 부를지, 어디에 배치할지 — 전부 사용자가 정한다.**
프로그램은 그 등록물을 실행해 줄 뿐이다. 한컴오피스는 그대로 쓰되, 반복 작업만 덜어낸다.

> 배경: 교사의 실제 업무(시험 출제, 가정통신문·공문 편집)에서 겪는
> "글씨·자간·표 양식이 매번 어긋나는" 스트레스를 없애려고 만들었다.
> 사람마다 자주 쓰는 기호도, 양식도, 서식 습관도 다르기 때문에
> **기능을 고정하지 않고 각자 채워 넣는 구조**로 설계했다.
> `30_exam_edit` → `exam_scribe` → **`hwp_palette`** 로 발전했다.

---

## 무엇을 저장할 수 있나

한글에서 만든 **결과물을 그대로 캡처**해 재사용한다. 한컴 매크로(과정 녹화)와 달리
결과를 통째로 보관하므로, 병합된 복잡한 표나 이미지가 든 표도 깨지지 않는다.

| 저장할 것 | 예 |
| --- | --- |
| 자주 쓰는 **문자·문구** | `√` `℃` `㉠`, 상용 인사말, 늘 쓰는 문장 |
| 내가 만든 **표·양식** | 결재란, 보기박스, 계획서 표지, 로고 박스 |
| 늘 쓰는 **서식 조합** | 굵게+빨강 / 발문용(내어쓰기+줄간격+글꼴) |
| 내가 정한 **호출 이름** | `\보기\` `\결재란\` `\인사말\` — 이름도 내가 지음 |

등록물은 **원본과 무관하게 프로그램 안에 복사**되므로 원본 파일을 지워도 그대로 남는다.
라이브러리 관리 창의 **내보내기/가져오기**로 항목을 zip 하나에 담아 동료와 주고받을 수
있다(조각 .hwp 파일까지 함께 들어간다). 폴더째 복사해도 물론 된다.

## 핵심 기능

### 1. 커스텀 팔레트 (탭 + 블럭)
메인 창은 고정 도구 모음이 아니라 **내가 채우는 빈 팔레트**다. 탭을 만들고
그 안에 블럭을 배치한다. 블럭은 클릭하면 실행되는 최소 단위이며 4종류다.

| 블럭 | 하는 일 | 예 |
| --- | --- | --- |
| **문자** | 커서 위치에 문자·문구 삽입 (1칸) | `√`, `℃`, 인사말 |
| **템플릿** | 문서 '일부'를 통째로 커서 자리에 삽입 (2칸) | 결재란, 보기박스 |
| **양식** | hwp 파일 '전체'를 새 문서로 열기 (2칸) | 계획서 표지, 가정통신문 |
| **서식 조합** | 선택 영역에 서식을 병렬 적용 | 굵게+글씨체+자간+크기 |

> **템플릿 vs 양식**: 템플릿은 *쓰던 문서에 꽂는 것*이라 용지·여백이 안 따라온다.
> 양식은 *그 양식으로 새로 시작하는 것*이라 용지·여백·머리말까지 그대로 열린다.

- **서식 조합 블럭 = 오토핫키식 병렬 조합**: 글씨체·자간·기울임·크기·색·정렬을 한 블럭에
  담아 클릭 한 번에 모두 적용.
- **환경설정**에서 탭을 추가/이름변경/삭제하고, 블럭을 **격자에서 드래그로 자유 배치**한다.

### 2. 마크다운 변환 (Ctrl+T)
한글에서 텍스트를 드래그하고 **마크다운 변환**을 누르면:

- **시험문제 문법** — `발문:` `질문:` `보기:` `선지:` 를 평가원 양식(문항 번호·〈보 기〉
  박스·① ② ③ 선지 표)으로 자동 조판.
- **라이브러리 문법** — `\라벨\` 로 등록한 문자·템플릿을 호출. 템플릿 속 빈칸 `\` 에는
  아랫줄 내용이 **위에서부터 순서대로** 채워진다. 비울 칸에는 `-` 한 줄.
  채우지 않은 빈칸은 자동으로 지워져 출력물에 `\` 가 남지 않는다.

```
\보기\
빛의 속력은 매질마다 다르다
굴절각은 입사각보다 작다
```
→ 보기박스가 삽입되고 ㄱ.ㄴ. 자리에 두 줄이 채워진다.

- **서식 적용** — `\명령{적용할 부분}` 으로 줄 일부에만 서식을 입힌다.
  구조는 LaTeX 에서 빌려왔다 (`\textbf{...}`) — 여는 글자 `\` 와 닫는 글자 `}` 가
  달라야 안에 다른 것을 넣어도 짝을 셀 수 있기 때문이다.

```
다음 중 \굵게{옳지 않은} 것은?
\굵게\기울임\크기15{나는 이제 할 수 있는게 없다.}
```

명령을 **원하는 만큼 앞에 쌓으면** 모두 적용된다(순서 무관). 쓸 수 있는 명령:

| 명령 | 뜻 |
| --- | --- |
| `\굵게` `\기울임` `\밑줄` | 토글 |
| `\15` `\크기15` | 글자 크기(pt) — 맨 숫자는 크기로 본다 |
| `\자간-5` | 자간 |
| `\색빨강` `\색#FF0000` | 글자색 |
| `\함초롬바탕` `\글꼴나눔고딕` | 글꼴 (목록 밖 글꼴은 `\글꼴` 을 붙여서) |
| `\내강조` | 라이브러리에 등록해 둔 서식 |

`{ }` 안에는 `\라벨\` 호출도, 빈칸 `\` 도, 다른 서식도 넣을 수 있다.
글자 그대로의 역슬래시가 필요하면 `\\` 로 쓴다.

> **`\라벨\` 과 `\명령{...}` 은 역할이 다르다.** 앞의 것은 등록해 둔 것을
> *꺼내 넣기*(붙여넣기), 뒤의 것은 그 부분에 *적용하기*(형광펜)다. 붙여넣기는
> '무엇을'만 있으면 되지만 형광펜은 '어디부터 어디까지'가 필요해서 모양이 다르다.

### 3. 라이브러리 (서식 · 문자 · 템플릿 · 양식)
한글에서 만든 결과물을 **그대로 캡처해 재사용**한다. 한컴 매크로(과정 녹화)와 달리
결과물을 통째로 저장하므로 병합된 복잡한 표도, 표 안의 이미지도 깨지지 않는다.

- **서식**: 원하는 항목만 골라 저장하는 부분 서식(“굵게+빨강”만 등)
- **문자**: 특수문자·상용구
- **템플릿**: 표·결재란 등 문서 일부 (`\` 로 빈칸 표시 → 나중에 채움)
- **양식**: hwp 파일 통째 — 파일만 고르면 등록(한글을 안 열어도 됨).
  클릭하면 새 문서로 열린다. 원본과 무관하게 복사되므로 원본을 지워도 남는다.

### 4. 양식 채우기 (AI와 주고받기)

양식 파일을 던지면 **채울 자리를 뽑아** 클립보드에 넣어준다. 그걸 Claude 같은 AI에
붙여넣고 "채워줘" 한 뒤, 받은 답을 다시 붙여넣으면 **원본 서식 그대로** 채워진
문서가 한글에서 열린다.

```
[1] \
[2] \. \.
[4] [\]
```

- **AI 호출은 이 프로그램이 하지 않는다.** 사람이 복사·붙여넣기로 나른다
  → API 키도, 네트워크도, 비용도 없다.
- 서식이 보존되는 원리: HWPX 는 zip + XML 이고 글자는 `<hp:t>` 태그에 들어 있다.
  **그 태그 내용만 바꾸고 나머지 바이트는 그대로 다시 쓴다.**
  실측(2026-07-19): 이미지 5개·셀 10개·병합 모두 보존, 한글에서 정상 열림.
- .hwp 는 한글을 통해 HWPX 로 바꾼 뒤 처리한다(바이너리 .hwp 는 직접 못 고친다).
- 한계: **글자만 갈아끼운다.** 행 추가 같은 표 구조 변경은 안 된다.

### 5. 사진 폴더 (\사진이름\)
라이브러리 창에서 사진 폴더를 연결해두면, 문서에 `\파일이름\` 이라고 쓰고
변환할 때 그 그림이 글자처럼 그 자리에 들어간다. 등록 절차 없이 **파일 이름이
곧 라벨**이다. 확장자는 안 써도 된다. 그림은 문서에 포함되므로 원본 폴더를
지워도 남는다. (우선순위: 등록 항목 > 내장 문자 > 사진 폴더)

### 6. 내장 문자
등록 없이 바로 쓰는 규칙적 기호. `\원1\`→①, `\로마3\`→Ⅲ, `\홑낫표\`→「」 등.
환경설정의 **내장** 탭에서 검색해 삽입할 수도 있다.

### 7. 양식 프리셋
시험지 표·박스·글꼴 스펙(2단/1단 폭, 셀 여백, 테두리 등)을 학교별로 이름 붙여
저장·전환하고 JSON으로 공유한다.

## 설치 · 실행

- **필요 환경**: Windows + 한컴오피스 한글, Python 3.10+
- pyhwpx(COM 자동화)로 실제 한글 프로그램을 원격 조종하므로 결과 서식이 완벽하다.

```bash
pip install -r requirements.txt
python main.py        # 또는 run.bat
```

한글을 미리 켜둘 필요는 없다 — 버튼을 누르면 자동으로 연결된다.

## 개발 메모

- 표·서식 자동화는 실제 한글 인스턴스로 실측 검증하며 만든다
  (WidthType 의미, 셀 여백 보정, 표 탈출 시퀀스, 블럭 캡처의 표-탈출 주의 등).
- 개인 데이터(config·library·fragments)는 커밋하지 않는다.
- 버전은 앱 하단과 이 문서에 표기하며, 의미 있는 완성 시점에만 올린다.
- 변경 이력은 [CHANGELOG.md](CHANGELOG.md)에 쌓는다. 릴리즈할 때 `[미출시]` 절을
  버전 번호와 함께 내리면 릴리즈 노트가 된다.
- 테스트: `python -m unittest discover -s tests` (한글 없이 실행된다)
